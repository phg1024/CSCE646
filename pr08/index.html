Project 08

Dithering and Screening

Method: Levels / BlockSize:

Pattern:

Image: Upload image:

Usage

  1. Choose the desired operation with the Method dropdown menu;
  2. Input a number of color levels (for dithering) / blocksize (for halftoning and pattern-based screening);
  3. Click on the Apply button to see the effect;
  4. For artistic screening using patterns, choose the pattern you want to use in the Patterns dropdown menu. Please make sure the block size is an even number, and use a block size larger than 4. Smaller block size are allowed but the results are not feasible.
  5. For comparison between the original image and the processed one, click and hold on the images to toggle the original images.
